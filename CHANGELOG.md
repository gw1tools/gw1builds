# What's New

## January 15, 2026

**Builder V2**

- Add equipment codes and configure individual armor, weapons, runes, and insignias
- Pick individual skills with search - see costs, recharge, and attribute-scaled descriptions
- Build variants let you save alternative skill bar setups (anti-caster, budget, etc.)
- Team builds now support up to 12 players with duplicate player slots
- Create new builds from existing ones as templates
- Collaborators can now edit your builds with you
- Private builds stay hidden from search but shareable via direct link
- Share draft builds before publishing
- Create builds without logging in first - sign up when you're ready to save
- Discord and email login options added
- Spotlight search (Cmd+K / Ctrl+K) searches both user builds and 26,000+ PvX Wiki archives
- Submit feedback, bug reports and feature suggestions via the feedback button

## January 12, 2026

- Notes editor now shows a live character count (and fixed a bug where character limits were counting JSON bytes instead of actual text)
- Nested lists, blockquotes, and dividers now render correctly in build notes
- Improved search experience on mobile devices

## January 11, 2026

- Search for builds by name, profession, or skills
- Quick access to PvX Wiki builds (external link) from the sidebar - Credits to PvX Wiki for all their incredible work
- Expanded content moderation and added a few more tags
- Fixed a few UI bugs

## January 9, 2026

**Initial Release**

- Build creation with skill bar and attribute distribution
- Profession selection (primary and secondary)
- Build notes with rich text editing
- Content moderation with reporting and profanity filtering
