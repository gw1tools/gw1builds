## Interactions

### Hover Lift
Cards and buttons lift up-left on hover:
- Transform: `translateY(-2px) translateX(-1px)`
- Shadow grows from Sticky â†’ Sticky Hover

### Press
Push down-right on click:
- Transform: `translateY(1px) translateX(1px)`
- Shadow shrinks to Sticky Active

### Focus
- Gold outline: `2px solid #e8b849`
- Offset: 2px

- Radius: Full (pill shape)

### Profession Badge
- Border: 2px solid, profession color
- Text: Profession color
- Small dot indicator in profession color
- Slight rotation

### Skill Slot
- Size: 48px mobile, 56px desktop
- Border: 2px solid
- Shadow: Sticky
- **Elite:** Gold border, gold-tinted shadow
- **Hover:** Lifts up, border turns gold

### Input Fields
- Background: Canvas (`#18181b`)
- Border: Default, solid
- Focus: Gold border + subtle gold ring
- Template inputs: Mono font, dashed border

### Cards
- Background: Surface (`#1f1f23`)
- Border: Default, solid
- Radius: Large (14px)
- Shadow: Sticky
- **Pinned variant:** Tape decoration on top
